Composer's/Author's name (100 $a)

This field contains the main entry for personal names. The main entry is created for the person responsible for the work. The composer (for music manuscripts and printed music), the librettist (for librettos) and the author of a work of music theory (for theoretica) can be considered the responsible persons.

The field is linked with the authority file "Names".

Dubious (100 $j)

If the authorship is uncertain, the box has to be ticked.
