Recurso externo (856)  

Use este campo para o link para um recurso externo que possui uma ligação directa com a fonte descrita. 

Recurso externo (856 $u) 

Insira o URL do recurso externo. Use sempre links permanentes (permalinks). 

Nota sobre o recurso externo (856 $z) 

Este campo é obrigatório quando se insere um link para um recurso externo. 

Insira uma breve descrição que explique porque o URL é relevante para a fonte descrita, usando a seu idioma de catalogação. 

Exemplos: 

Cópia digital 

Marca de água na p. 4 

Homepage ou página do projecto 

Detalhe da encadernação  

Registo/registro bibliográfico 

Link para o registo/registro em Bach Digital  

Link para a obra no Frescobaldi Thematic Catalogue Online 

Entrada nos registos/registros da igreja 

Link para RISM ID no. 806155758, GB-Lbl Add. 14209 f.23r-27v, um setting deste texto por N. Porpora   

Fonte concordante em GB-Ob 

Tipo de link (856 $x) 

Este campo é obrigatório quando se insere um link para um recurso externo. Seleccione dos seguintes: 

  • Fonte digitalizada: O link é para um website externo que é uma cópia digitalizada do recurso a ser descrito. Os links para repositórios institucionais são preferenciais mas se um não estiver disponível, então links para repositórios externos, como o Internet Archive ou o IMSLP são permitidos. 

Exemplo: 

https://mirador.acdh.oeaw.ac.at/musikarchivspitz/A-SPD_A15/ 

  • Manifesto IIIF: O objecto ligado é um objecto JSON legível por máquina, processado por um visualizador de documentos interno como o diva.js. O documento está embebido directamente na página web. Muitas das vezes, “manifest”, “iiif” ou semelhante consta no link. 

Exemplo:  https://iiif.lib.harvard.edu/manifests/drs:2820650 

Nos casos em que ambos os links para um visualizador externo e um manifesto IIIF estiverem disponíveis, repita o campo e liste ambos os links separadamente 

Exemplo: 

  • Recurso externo: http://nrs.harvard.edu/urn-3:FHCL.Loeb:537966 

  • Nota: versão digitalizada 

  • Tipo de link: fonte digitalizada 

  • Recurso externo: https://iiif.lib.harvard.edu/manifests/drs:2820650 

  • Nota: manifesto IIIF  

  • Tipo de link: manifesto IIIF 

  

  • Outros: Links para outros recursos que não correspondem ao recurso que está a ser descrito. 

  

Tipos de recursos externos 

Tipo de link 856 $x: Fonte digitalizada Link apenas para a fonte descrita no registo/registro RISM. Use apenas links permanentes (permalinks). Se os links permanentes estiverem indisponíveis, pode criar um link para uma página onde o link para a fonte digitalizada pode ser facilmente encontrado (como um registo/registro bibliográfico num catálogo de biblioteca). Pode incluir partes da fonte digitalizada (como algumas páginas) se uma versão totalmente digitalizada não estiver disponível. 

Links inseridos como música digitalizada aparecem no filtro "Música digitalizada" no OPAC público (opac.rism.info). 

Tipo de link 856 $x: Outro 

Outros tipos de recursos externos podem incluir os seguintes. Certifique-se de que é sempre claro para o utilizador porque se está a incluir um link, como na Nota geral (500). 

  • Detalhes de fontesDetalhes de fontes que apresentem apenas alguns aspectos, como marcas de água, encadernações, título de página, ou capa. 

  • Registos catalográficos/Bases de dados Inclui entradas bibliográficas em catálogos externos ou entradas em bases de dados externas. Bases de dados académicas externas que forneçam informação útil podem ser incluídas, mas considere se poderão estar melhor inseridas como um link para literatura secundária.  

  • Catálogos de obras electrónicos,  enciclopédias online, fontes de referência digitalizadas De um modo geral, use os campos Literatura Secundária (691) ou Catálogo de obras (690) para se referir a obras de referência online, como catálogos de obras ou enciclopédias. No entanto, como um serviço para os utilizadores, pode ser útil incluir adicionalmente um link directo para um determinado local dentro do recurso, através do campo 856. 

  • Fontes arquivísticas, fontes históricas Inclui registos/registros ou documentos municipais digitalizados, correspondência digitalizada, jornais históricos. Assegure-se de que a relevância da fonte arquivística é tornada clara numa Nota geral (500), então inclua o link directo aqui. Para jornais históricos, deve inserir o nome do jornal como literatura secundária mais pode inserir o link para uma página específica onde se encontra o artigo aqui.; certifique-se que informação sobre o artigo, como o título e a data, é incluída numa nota. 

  • Websites Inclui websites de projectos, websites de agências de financiamento externas, ou outros websites relevantes.  

  • Outros registos/registros RISM Link para outro registo/registro RISM apenas se o registo/registro tiver uma relação directa com a fonte descrita. Use apenas links permanentes (permalinks) do RISM OPAC (https://opac.rism.info/). Deve ser claro para o utilizador porque o registo/registro é relevante para a fonte. No mínimo, nomeie a instituição detentora, cota ou código de referência, e número de RISM ID numa nota. Por vezes, a ligação para uma fonte diferente pode ser melhor explicada numa Nota geral (500).  Se fizer referência a outros registos/registros RISM noutro local do seu registo/registro, não é necessário criar um link para eles neste campo. Não é necessário criar um link aqui para todas as fontes concordantes no RISM; na verdade, essa é uma busca infrutífera porque o número de fontes no Muscat cresce diariamente.  

  • Referências a fontes relevantes noutras instituições, mas não no RISM Se quiser criar um link para uma fonte que não esteja ainda no RISM, considere se é possível que seu grupo de trabalho insira primeiro o libreto, manuscrito ou tratado digitalizado, ou adicione existências de um impresso específica. Os Serviços Centrais terão prazer em ajudá-lo se acha que uma fonte deve estar no RISM. No entanto, é compreensível se não conseguir criar um novo registro para uma fonte primeiro. 

  • Outras fontes relevantes Inclui fontes concordantes, obras concordantes, árias ou seções de obras concordantes, fontes que serviram de base para a fonte que está a ser descrita. Esses links podem ser para os objetos digitalizados ou links para catálogos externos.
