Campo obbligatorio.

Questa parte registra la data d’origine di una fonte e anche qualsiasi data che compare sulla fonte. Ripetendo il campo, fai attenzione che nella scheda risulti chiaramente (ad esempio in un campo note) a cosa si riferisce ciascuna data. 

 

Tipo di datazione (033)

Seleziona il tipo di datazione:

  • Data singola
  • Date singole multiple
  • Ambito di datazione

 

Data (033 $a)

Inserisci diversi tipi di date utilizzando la forma codificata AAAAMMGG. Singoli anni possono essere indicati AAAA.

  • Data singola: AAAAMMGG
  • Date singole multiple: AAAAMMGG, AAAAMMGG, AAAAMMGG, etc.
  • Ambito di datazione: AAAAMMGG-AAAAMMGG

 

Esempi:
Data singola: 1769
Data singola: 18190913
Date singole multiple: 18791213, 18791216, 18791223

 

Usa un trattino se alcune parti della data sono sconosciute o mancanti.

Esempio per dicembre 1791:
179112--

Se hai utilizzato ca., before o after nel campo Data (260 $c), inserisci per favore almeno un anno in questo campo. Utilizza al meglio le tue risorse per determinare razionalmente un ambito di datazione; ad esempio, puoi usare le filigrane o le date di nascita e di morte del compositore per restringerlo. Ambiti di datazione più stretti, all’incirca di vent’anni, sono più utili che ambiti più vasti.
