Perguntas frequentes (em português)

Usando o Muscat

  1. Toda a literatura citada no Muscat está no Escritório Central?
    Não, somente a literatura rotulada "HB" ou "Handbibliothek" ou "RISM-ZR" está de fato na biblioteca do Escritório Central. Estes materiais são para benefício de todos os contribuidores do RISM, de maneira que se temos uma publicação que você gostaria de consultar, informe-nos de tentaremos obter a informação de que você precisa. Mesmo se você tiver interesse em uma publicação que não temos no escritório, existe uma chance de que conheçamos alguém que a tenha.

  2. O que devo fazer se esqueci minha senha ou se quero modificá-la?
    As senhas são administradas pelo Escritório Central. Entre em contato conosco se perdeu sua senha. Não é permitido modificar a sua senha.

Aspectos técnicos do Muscat

1. Quais são os requisitos técnicos para o Muscat?
  • O Muscat é independente de plataforma e trabalha tanto em Macs como em PCs.
  • O acesso é feito através de uma URL e requer uma conexão à internet.
  • O Muscat trabalha melhor com telas que tenham pelo menos 1366 x 768 pixels de resolução.
  • O Muscat está otimizado para Firefox e Chrome. Não use o Internet Explorer!

2. Alguns aspectos técnicos sobre o Muscat.

  • O Muscat é “open source”, código aberto. O código fonte está disponível no repositório GitHub.
  • O Muscat é uma aplicação Ruby on Rails.
  • Verovio é usado para representar incipits musicais através de MEI.
  • Solr é usado como um motor de busca.
Mais informação sobre o desenvolvimento do Muscat pode ser encontrada no website do RISM Suíça.


3. Algumas características do Muscat.

  • Versões: Catalogadores podem ver as modificações feitas nos registros.
  • VIAF: Nomes de pessoas podem ser importados através do Virtual International Authority File (VIAF). 


4. Quais são os planos para futuros aprimoramentos do Muscat?

  • Integrar images usando IIIF
  • Coletar feedback do usuário.
  • Acrescentar traduções adicionais da interface e das diretrizes.
  • Acrescentar uma busca por incipit.
  • Importar/exportar registros em lote para o Muscat e do Muscat para outros sistemas.
