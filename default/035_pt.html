Número Local  (035)

Use este campo para registrar um número para uso interno, como os números RISM de nove dígitos ou outros números locais (números OCLC). Este campo é publicado no catálogo on line.

Para o projeto Access to Music Archives / RISM Series C:

GND code (MARC 035)

Insira a informação do identificador [identifier information] de acordo com o Registro Integrado de Autoridade (em alemão, “Gemeinsame Normdatei”, ou em inglês, “Integrated Authority File”).
