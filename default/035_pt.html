Número Local  (035)

Use este campo para registrar um número para uso interno, como os números RISM de nove dígitos ou outros números locais como os números de controle OCLC.  Especifique de que tipo de número se trata. Este campo é publicado no catálogo on line.

Exemplo:
(OCoLC)891760516

Para o projeto Access to Music Archives / RISM Series C:

GND code (MARC 035)

Insira a informação do identificador de acordo com o Registro Integrado de Autoridade (em alemão, “Gemeinsame Normdatei”, ou em inglês, “Integrated Authority File”).
