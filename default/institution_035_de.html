GND Code (MARC 035, Local number)

Geben Sie hier den GND Code für eine Bibliothek oder Körperschaft ein (GND = "Gemeinsame Normdatei"). Es können mehrere Codes für eine Körperschaft vorhanden sein. Das Eingabefeld ist verdoppelbar. Der Code/ Die Codes können der Webseite der Deutschen Bibliothek entnommen werden: http://http//www.dnb.de/
