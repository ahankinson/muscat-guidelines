Nome de pessoa adicional  (700)

Este campo aparece duas vezes para fontes: na seção Pessoas e instituições, para listar todas as pessoas envolvidas com a fonte, e na seção Descrição material, para associar nomes de pessoas adicionais a grupos de materiais particulares, especialmente ao criar múltiplos grupos de materiais.

Nome (700 $a)

Insira todos os nomes de pessoas que aparecem na fonte. Este campo é relacionado ao registro de autoridades Nomes de pessoas. Nomes que ainda não estejam no registro de autoridades podem ser adicionados. 

Editores nomeados como pessoas mas que atuem na função de editor (Pleyel, André, etc.) são inseridos como Instituição adicional (710).

No caso de registro principal para coletâneas ou volumes compostos, insira um nome somente se este nome se aplica a todas as obras na coletânea. Insira nomes associados com obras individuais nos registros individuais.

Atribuição (700 $j)

Selecione um indicador para descrever o nível de certeza da atribuição se qualquer uma destas situações se aplica. Caso contrário, deixe em branco..

  • Suposta significa que o nome está presente na fonte, mas a atribuição é questionável ou duvidosa.
  • Confirmada significa que o nome está parcialmente presente ou totalmente ausente na fonte, mas a atribuição é (razoavelmente) certa e baseada na comparação com fontes concordantes, literatura secundária ou outros recursos acadêmicos. É o que ocorre na maioria dos casos.
  • Conjetural é uma atribuição estimativa, baseada em evidência secundária (tal como literatura secundária).
  • Duvidosa significa que a autoria está em questão.
  • Equivocada significa que o nome está na fonte, mas a atribuição é incorreta.


Função (700 $4)

Campo obrigatório se “Nome” for preenchido.

Atribua a cada pessoa uma função exata. Uma pessoa pode ser registrada várias vezes com diversas funções. Uma lista reduzida de funções é oferecida na informação de acervo para itens impressos. Os códigos de três letras se referem ao código MARC para relatores.

  • Arranjador [arr]
  • Autor [aut]: A pessoa responsável pela criação de uma obra que é basicamente textual no conteúdo, como um livro.
  • Censor [cns]
  • Co-compositor [ctb]: Use para compositores que também contribuíram musicalmente para a obra.
  • Referência cruzada de compositor [cmp]: Use para nomear um compositor que é ou foi associado à peça de alguma maneira. Exemplos são: um compositor a quem a obra é ou foi incorretamente atribuída, um compositor cuja melodia é usada em um conjunto de variações.
  • Conceptor [ccp]: A pessoas responsável pela ideia original na qual se baseia uma obra, tal como o autor de um livro ou peça em que uma ópera foi baseada.
  • Copista [scr]: Use para copista ou escriba.
  • Dedicatário(a) [dte]: Uma pessoa à qual a fonte é dedicada.
  • Distribuidor [dst]
  • Editor [edt]
  • Gravador [egr]
  • Proprietário anterior [fmo]
  • Ilustrador [ill]: A pessoa que contribui para uma fonte com suplementos ao conteúdo principal como desenhos, diagramas, fotografias, etc. 
  • Libretista [lbt): Autor do libreto de uma ópera, de outra obra cênica ou oratório.
  • Litógrafo [ltg]
  • Outro [oth]: Todos os demais nomes e funções, incluindo importadores, distribuidores e outros contribuidores.
  • Intérprete [prf]: Qualquer intérprete, como um instrumentista, vocalista, músico, regente, dançarino, ou se a habilidade principal é desconhecida ou não especificada.
  • Impressor [prt]
  • Publicador [pbl]: Somente insira o publicador como uma pessoa se a própria pessoa for mencionada, como em um manuscrito de Beethoven que menciona Carl Friedrich Peters pelo nome (Peters funcionando como a companhia que lançou a música impressa, por outro lado, é inserida como uma instituição.) 
  • Autor literário [lyr]: Um autor das palavras de uma obra musical não-dramática (como o texto de uma canção), exceto para oratórios.
  • Tradutor [trl]
