Notas interna (599 $a)

Utilice este campo para registrar notas locales para uso interno, inclusive dentro de su grupo de catalogación. Puede utilizarse cualquier idioma para escribirlas. Estas notas no se publican en el catálogo público online. Las mismas pueden utilizarse para comentarios de todo tipo, como preguntas, comentarios o apuntes de indagaciones pendientes. Borre las notas después de que hayan cumplido con su fin. Es una buena práctica firmar su nota con sus iniciales y la fecha.
