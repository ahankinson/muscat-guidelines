Use this field to link to an external resource. The preference is to link to institutional repositories but if one is not available then links to external repositories such as the Internet Archive or IMSLP are allowed.


External resource (856 $u)

Enter the URL of the external resource. Always use permalinks.


Note about external resource (856 $z)

This field is required when entering a link.

Enter a brief note, such as "Digital copy" or the page number of the source being described. This note is published in the catalog and helps users understand what the link is. Enter using your cataloging language.


Link type (856 $x)

This field is required when entering a link to an external resource. Select from the following:

  • Digitized music: The link is to an external website which is a digitized copy of the resource being described in the current RISM record.
    Example    :
    https://mirador.acdh.oeaw.ac.at/musikarchivspitz/A-SPD_A15/
  • IIIF manifest: The linked object is a machine-readable JSON object processed by an internal document viewer such as diva.js. The document is embedded in the web page directly. In many cases, "manifest," "iiif," or similar appears in the link.
    Example    :
    https://iiif.lib.harvard.edu/manifests/drs:2820650
  • Other: Link to other resources that are not the resource being described.

In cases where both links to an external viewer and an IIIF manifest are available, repeat the field and list both links separately.
Example:

  • External resource: http://nrs.harvard.edu/urn-3:FHCL.Loeb:537966
    Note    : digitized version
    Link type    : digitized music
  • External resource: https://iiif.lib.harvard.edu/manifests/drs:2820650
    Note    : IIIF manifest
    Link type    : IIIF manifest
