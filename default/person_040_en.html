Cataloging source (040)

Do not change.

This field records the organization that created the original record. For new records, this is always RISM, so the ISIL code for RISM is automatically filled out: DE-633.

The Staatsbibliothek zu Berlin – Preußischer Kulturbesitz is responsible for maintaining the codes for Germany. See http://sigel.staatsbibliothek-berlin.de/suche/?isil=DE-633



Original cataloging agency (040 $a)

This field records the code for the organization that created the original record. The code for RISM is automatically filled out by Muscat.

 

Transcribing agency (040 $c)

This field records the code for the organization that transcribed the record into machine-readable form. The code for RISM is automatically filled out by Muscat.


Modifying agency (040 $d)

Do not change.

This field records the code for the organization responsible for modifying the MARC record. Canadian, British, and German codes can be searched here: https://www.loc.gov/marc/organizations/other_assignment.php Codes for all other countries can be searched here: https://www.loc.gov/marc/organizations/org-search.php


Language of cataloging (040 $b)

Do not change.

This field records the MARC code for the language of the catalog for which the record is intended. The code ger is always automatically filled out.
