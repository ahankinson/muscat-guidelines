Catalogação de coletâneas

No RISM, uma coletânea é tanto um manuscrito coletivo intencional ou antologia de múltiplas obras assim como um item que contém duas ou mais peças separadas. O segundo significa que se um manuscrito de uma sonata para piano, por exemplo, também tiver um esboço para uma fuga no final, o RISM o catalogaria como uma coletânea com um registro principal para o item como um todo e registros secundários (entradas individuais) para a sonata e a fuga.

Ao catalogar coletâneas, por favor observe que:

  1. O registro principal (entrada principal) da coletânea deve conter informação que se aplica à coletânea como um todo.
  2. Qualquer informação que seja relevante para alguma parte da coletânea deve ser inserida no registro secundário (entrada individual).

Se as obras na coletânea têm compositores diferentes, diferentes formações, tipos de fonte etc., então não insira tais dados no registro principal.

Para coletâneas em que as obras são muito similares, é uma boa estratégia criar um primeiro registro completo e então usar a função “Duplicar” para copiar o registro. Então para os registros seguintes você tem que modificar apenas como título e número de página.

Registros principais de coletâneas devem ser criados usando o modelo Manuscrito musical, registro principal de coletânea para itens nos quais (1) os tipos de fontes são mistos, tais como itens manuscritos e impressos encadernados juntamente, ou (2) múltiplas edições impressas que foram encadernadas juntamente. Observe o seguinte:
1. Deixe o campo Tipo de fonte (593) em branco se os materiais são mistos.
2. Uma breve descrição sobre a natureza do item pode ser adicionada como uma nota.
3. Use o modelo apropriado para manuscrito ou impresso para os registros secundários.
4. Para material impresso, vincule o item ao registro principal através do registro do material. Vincule o campo Encadernado com (973).
5. Manuscritos musicais são vinculados a registros principais da maneira usual (campo 773).
(Diretrizes para isto serão ainda desenvolvidas.)

Registros principais de coletâneas não devem conter registros principais de coletâneas de manuscritos. Ao invés disso, use uma hierarquia mais simples por meio de registros secundários e explique a estrutura, a composição ou a organização do item em uma nota.

Catalogação de pastiches e compilações

Pastiches e compilações são catalogados como coletâneas. Cada seção é inserida separadamente como um registro secundário em uma coletâneas. Diretrizes mais detalhadas virão depois.

Catalogação de arranjos para piano

Especifique arranjos para piano da seguinte maneira:

  • No campo Título uniforme (240):

    • Selecione Arranjo ao lado de Definição de arranjo..

    • Em Síntese da formação, insira a formação do arranjo, tal como pf ou V, pf. Não insira a formação original da obra.

  • No campo Formação total (594), insira a formação total do arranjo. Não insira a formação original da obra.

  • No campo Descrição física (300), use keyboard score se a obra foi reduzida para piano. Use vocal score se a obra foi reduzida para voz e piano.
