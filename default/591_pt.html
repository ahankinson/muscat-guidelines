Outro código (591)

Insira outros códigos, cotas ou números (também chamados de “números de chamada”) aqui. Quaisquer códigos, cotas ou números atualmente válidos devem ser inseridos neste campo. Por exemplo, algumas óperas têm cotas separadas para cada ato, ou alguns livros de partes de coro têm cotas individuais para cada livro de coro. Entre com os códigos, cotas ou números da maneira indicada para o campo Códigos (852 $p) acima.

Códigos, cotas ou números que não estão mais em uso devem ser inseridos no campo Código antigo (olim) (852 $d).
