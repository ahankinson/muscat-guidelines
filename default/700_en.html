Additional personal names (700)


This part records all additional people involved, in the broadest sense, as authors of a work. This part also records all other people named on the source.

Name (700 $a)

Enter all personal names that appear on the source. This field is linked to the index People. Names not yet in the authority file may be added. 

In the case of parent records for collections or composite volumes, enter a name only if the name applies to all the works in the collection. Enter names associated with individual works in the individual records.

Attribution (700 $j)

Required field if "Name" is filled out.

Select an indicator to describe the level of certainty with the attribution:

  • Verified means the person's full name appears on the source. This happens rarely.
  • Ascertained means the name is partially present or entirely absent from the source, but attribution is (reasonably) certain and based on comparison with concordant sources, secondary literature, or other scholarly resources. This happens in most cases.
  • Conjectural is a guess based on secondary evidence (such as secondary literature).
  • Doubtful means authorship is in question.
  • Alleged means the name is present on the source, but the attribution is questionable or doubtful.
  • Misattributed means the name is on the source but is incorrect.

Function (700 $4)

Required field if "Name" is filled out.

Assign each person an exact function. A person can be recorded several times with various functions.

  • Authors
    • Composer/co-composer: Use also for co-composers and composer cross references.
    • Arranger
    • Text author: An author of the words of a non-dramatic musical work (such as the text of a song), except for oratorios
    • Librettist: Author of the libretto to an opera or other stage work, or oratorio
    • Translator
    • Compiler: The person responsible for creating a new work through the act of compilation.
    • Conceptor: The person responsible for the original idea on which a work is based, such as a play that served as the basis for an opera.
    • Author: The person responsible for creating a work that is primarily textual in content, such as a book.
    • Contributor: The person responsible for making contributions to the source. Use if a more specific function is unavailable or unknown.
  • Artistic contributors
    • Instrumentalist
    • Vocalist
    • Conductor
    • Dancer
    • Performer: Any other performer, or if the principal skill is unknown or unspecified.
    • Choreographer
    • Artist: The person responsible for an original graphic design, drawing, painting, etc. Includes stage designers.
    • Costume designer
    • Production personnel: The person associated with the production (props, lighting, special effects, etc.).
    • Censor
  • Technical contributors
    • Copyist/scribe
    • Publisher
    • Printer
    • Engraver
    • Lithographer
    • Illustrator: The person contributing to a resource by supplementing the primary content with drawings, diagrams, photographs, etc. If the work is primarily the artistic content created by this person, use Artist (in the group Artistic contributors).
    • Editor
    • Binder: Includes bookbinders.
  • Other
    • Dedicatee
    • Former owner
    • Donor: A former owner of an item who donated that item to another owner.
    • Patron: A person responsible for commissioning a work.
    • Bookseller
    • Event organizer
    • Other: Use if the exact function is not on the list.
    • Not known: Use if the exact function is unknown, unclear, or unspecific.
