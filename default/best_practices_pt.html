Boas práticas no RISM

1. Campos de texto livre e campos indexados
Se você menciona uma pessoa ou instituição em seu registro (como em um campo de notas), certifique-se de também inserir o nome nos campos indexados Nome de pessoa adicional (700) ou Instituição adicional (710).

O mesmo se aplica ao citar literatura secundária: se você menciona uma publicação em um campo de notas, certifique-se de também inseri-la no campo Referência bibliográfica (691).

2. Citando literatura secundária
Informação que não é encontrada na fonte pode ser incluída no registro RISM mas uma Referência bibliográfica (691) deve ser adicionada. Por exemplo, se o compositor de um manuscrito é conhecido através de um livro, ou se o ano de publicação para um impresso é dado em um catálogo, a fonte da informação deve ser claramente informada através de uma nota (500) e de uma referência indexada (691).

3. URLs e links para recursos externos
Sempre use permalinks ao criar links para música digitalizada, em particular para o campo Recurso externo (856)).

Se você quer criar um link para um sítio web diferente, não insira o URL em um campo de notas. Ao invés disso, insira o recurso como uma referência bibliográfica inserindo-o na base de dados Literatura secundária. Então, faça um link para ele no campo Referência bibliográfica (691) e indique-o pelo seu título curto em um campo de notas. Isto fornece aos usuários acesso à informação bibliográfica completa e, se o URL muda, então temos apenas que modificá-lo uma única vez no registro bibliográfico.

Para livros encontrados no Google Books ou em outros repositórios online, sempre crie um  registro bibliográfico na base de dados Literatura secundária para o livro e não para o repositório; i.e. cite o livro e não o Google Books.


4. Fazendo identificações
Informação importante como nomes de compositores ou títulos de obras não estão sempre mencionada na fonte. Às vezes é possível fazer identificações com base em literatura secundária, fontes de referência, outros registros RISM, ou mesmo de fontes de arquivo a que você tenha acesso. Se você recorrer a esses recursos, sempre deixe claro no registro o que você usou para fazer a identificação: nomeie o recurso em uma Nota geral (500) e adicione uma Referência bibliográfica (691), se apropriado. Certifique-se de que esteja claro para os usuários posteriores do RISM como você chegou às suas conclusões.
