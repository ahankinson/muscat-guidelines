Name variant (400)

This field records forms of the name that differ from the authorized form, such as birth names, pseudonyms, religious names, names spelled differently on different sources, alternate transliterations, etc.

Name variant (400 $a)

Enter the name in the form described in the field Heading - Personal name (100 $a).

For names originally written using non-Latin scripts, enter the form in the non-Latin script in this field. Transliterations using other translation tables can also be entered here.

Additions to names such as Jr, III, Sir, Meister,etc. are entered here, for example: Sinatra, Frank Jr.


Life dates (400 $d)

Do not enter anything in this field.


Type of name variant (400 $j)

Select from the list:

    • Baptismal name
    • Birth name
    • Byname: This is for nicknames
    • Initials
    • Married name
    • Order name: This is for religious names
    • Pseudonym
    • Other form: This is for alternate spellings, transliterations, or names using non-Latin alphabets
    • Translation: For example, Ackermann, Alexander is a translated variant of Agricola, Alexander
    • family name: Do not use
    • monogram: Do not use
    • other searchform: Do not use
    • person name: Do not use
    • unknown: Do not use
