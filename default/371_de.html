Adresse (Marc 371) (ISDIAH 5.2.1)

In den Eingabefeldern zur Adresse einer Bibliothek oder Körperschaft können sowohl physische, als auch elektronische Adressangaben gemacht werden. Der gesamte Eingabeblock ist verdoppelbar sowie jedes einzelne Eingabefeld.

Folgende Eingabefelder stehen zur Verfügung:

  • Straße und Hausnummer (371 $a)
  • Additions to the address: Storeys (for example, 1st floor, room 569), Names of buildings (for example, Palazzo ducale, Rose cottage etc.) [MARC field?]
  • Postleitzahl (371 $e)
  • Ort (371 $b)
  • Untergeordnete Gebietskörperschaft (371 $c)
  • Staat oder Land (371 $d)
  • URL (371 $u): Hier wird die offizielle URL der Bibliothek oder Körperschaft eingetragen. Existiert keine eigene Webseite, so kann die URL der übergeordneten Insitution angegeben werden. 

Die Adressangaben werden nach den Konventionen der jeweiligen Länder gemacht. 


E-Mail-Adresse (MARC 371 $m; ISDIAH 5.2.2)

Hier wird die E-Mail-Adresse der Bibliothek oder Körperschaft eingetragen. 


Weitere Kontakdaten (MARC 371 $z; ISDIAH 5.2.2, 5.2.3)

  • Telefonnummer (ISDIAH 5.2.2)
  • Faxnummer (ISDIAH 5.2.2)
  • Kontaktperson  (ISDIAH 5.2.3): Hier kann der Name einer Kontaktperson eingetragen werden. Der Name der Kontaktperson wird nicht veröffentlicht. 
  • E-Mail-Adresse der Kontaktperson (ISDIAH 5.2.3) wird veröffentlicht.
